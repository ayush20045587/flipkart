# 🛒 Flipkart Shopping App

A simple **shopping cart web app** built with **Python, Streamlit, and Pandas**.
Users can view products, add items to a cart, and generate an invoice with **discount and GST calculation**.

## 🚀 Features

* View available products
* Add/remove items from cart
* Automatic stock update
* Invoice generation
* 10% discount for orders above ₹50,000
* 18% GST calculation

## 🛠 Tech Stack

* Python
* Streamlit
* Pandas

## ▶️ Run the App

Install dependencies:

```bash
pip install streamlit pandas
```

Run the application:

```bash
streamlit run main.py
```
